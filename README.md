#Song Bot

##Overview

Song Bot helps you to download bollywood songs right from your terminal. 

It was becoming a big pain to navigate around for downloading a single song, and hence created this
bot.

##Installing

```pip install -r requirements.txt```

##Running 

```python song-bot.py```
